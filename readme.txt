== Installation ==



This section describes how to install the plugin and get it working.



1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/ directory



2. Activate the plugin



3. You will see "Professional LinkedIn Widget" settings from right side of your wordpress menu.



Configuration Options:



LinkedIn Public Profile ID :Default is "linkedin" - You can change it with your profile id



Margin Top :Their you have to Put Numeric Number as per your need. Margin Top create a padding from Top position of your website screen
